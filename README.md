# CodeAlpha-EDA-Visualization
Exploratory Data Analysis and Data Visualization using Python

## 📌 Project Overview
This project focuses on performing Exploratory Data Analysis (EDA) on a dataset obtained from web scraping.

EDA helps in understanding the structure, quality, and patterns in the dataset before applying advanced analytics or machine learning.

## 🎯 Objectives
- Inspect dataset structure
- Identify data types
- Check for missing values
- Analyze frequency distribution
- Extract meaningful insights

## 🛠 Technologies Used
- Python
- Pandas
- Matplotlib

## 📂 Project Files
task2_eda.py  
quotes_dataset.csv  

## ▶️ How to Run

1. Install required libraries:
pip install pandas matplotlib

2. Run the script:
python task2_eda.py

## 📊 Analysis Performed
- Displayed first 5 rows
- Checked dataset information
- Verified missing values
- Counted quotes per author
- Generated author frequency bar chart

## 🔍 Key Findings
- Dataset contains no missing values
- Some authors contribute multiple quotes
- Data is clean and ready for visualization
